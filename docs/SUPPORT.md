# Support Guidelines

Welcome to our Support page! Before opening an issue or asking a question, please review the resources below to get help as quickly as possible.

*Chào mừng bạn đến với trang Hỗ trợ! Trước khi mở một issue hoặc đặt câu hỏi, vui lòng điểm qua các tài nguyên bên dưới để được trợ giúp nhanh nhất.*

---

## 1. Frequently Asked Questions (FAQ)

Before seeking personal support, please check if your question is answered below:

*Trước khi tìm kiếm hỗ trợ trực tiếp, vui lòng kiểm tra xem câu hỏi của bạn có nằm dưới đây không:*

* **Q: How do I install Feather? 

/ Làm sao để cài đặt Feather?**
  * **A:** Please use TrollStore or AltStore to install the latest `.ipa` file from our Releases page. 

/ Vui lòng sử dụng TrollStore hoặc AltStore để cài đặt file `.ipa` mới nhất từ trang Releases.*
* **Q: Sileo shows "Verification Timed Out" 

/ Sileo báo lỗi hết hạn xác thực?**
  * **A:** Check your network connection or try switching from Wi-Fi to Cellular data. 

/ Kiểm tra lại kết nối mạng hoặc thử chuyển từ Wi-Fi sang mạng di động.*

## 3. Opening a GitHub Issue

If you have completed the steps above and still experience a bug with the software itself, you may open an issue using our provided templates. 

*Nếu bạn đã làm theo các bước trên và vẫn gặp lỗi trực tiếp từ phần mềm, bạn có thể mở một issue bằng các biểu mẫu có sẵn.*

* **DO NOT** use issues to ask for general help. / 
*KHÔNG dùng issue để hỏi đáp thông thường.*

* **DO NOT** report security vulnerabilities here (use `SECURITY.md`). / 
*KHÔNG báo cáo lỗ hổng bảo mật tại đây (hãy xem `SECURITY.md`).*
